# Project-1-Banking-with-Java

## Project overview
this project is creating a simple command-line banking application.
the application has a simple authentication flow
account management and transaction processing.
the system is also designed with fraud detection mechanisms and traceability.

## The technologies used are:
* Java
* Lucid Chart for ERD diagramming
* Trello for Project management and tracking
